# Sistema de Gestión de Reservas - C.A.F.

Este proyecto consiste en el desarrollo de un sistema informático de gestión de reservas para el gimnasio *C.A.F.*. El sistema permite a los usuarios registrar, modificar, cancelar y confirmar sus reservas de sesiones, utilizando su correo institucional y su RUT. 

## Características principales:
- Optimización del cupo diario disponible.
- Accesible desde cualquier dispositivo como una aplicación web.
- Interfaz intuitiva que facilita la planificación y gestión del tiempo de los usuarios.

##Requisitos previos

-Ionic_CLi
